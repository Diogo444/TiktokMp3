# TikTok MP3 - Frontend

Application Vue.js pour télécharger et convertir des vidéos TikTok en MP3.

## Installation

```bash
pnpm install
```

## Développement

```bash
pnpm dev
```

L'application sera accessible sur http://localhost:5173

## Build

```bash
pnpm build
```

## Preview

```bash
pnpm preview
```

## Technologies

- Vue 3 avec `<script setup>`
- Vite
- JavaScript ES6+
